If you are buying a GameCube today, not every unit is equally useful. Some revisions are much better than others depending on whether you care about video quality, homebrew, or modern storage options.

## The Main Model to Look For

The best model to target is generally the **DOL-001**.

Why?

- it has the **digital AV out**
- that opens the door to component video
- it also enables HDMI and VGA adapter options that do not work on later models

If video quality matters, this is the version to prioritize.

## Serial Port 2 Matters Too

Some DOL-001 consoles also include **Serial Port 2**. That makes the console even more attractive because it can be used with:

- an **SD2SP2** for homebrew loading
- some GameCube network adapter projects
- convenient storage setups that do not occupy the memory card slot

Not every DOL-001 has this extra port, so it is worth checking photos carefully before buying.

## If Your Console Lacks Serial Port 2

That is not a deal-breaker. You can still use an **SD Gecko** style memory card adapter to load software from SD storage.

The downside is convenience:

- the adapter takes up a memory card slot
- it is less tidy than SD2SP2
- swapping between storage and real memory cards can be annoying

## Region Considerations

A **US GameCube** is often the nicest option if you care about progressive scan support, because US software tends to support it more often.
