Forget BVM/PVMs, Multisync Monitors are Great

  

Cheaper and Better than a PVM


This Microvitec CRT monitor supports 15kHz RGB natively, so there’s no need for line doubling or deinterlacing. But I do need a sync stripper to fix the composite sync that the PS2 outputs. For this I’m using a Sync Strike:

  

http://arcadeforge.net/Scaler-and-Strike-Devices/Sync-Strike::15.html?language=en

  

So the connection is:

  

1. PS2 output via RGB SCART

2. RGB SCART to 15kHz VGA via Sync Strike

  

Hope this helps!
