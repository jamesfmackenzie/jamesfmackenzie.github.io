There was a brief era where Microsoft gaming services extended into phones and portable devices in a surprisingly interesting way.

This draft is based on the idea that Games for Windows Live and Windows Mobile made it possible to keep earning achievements away from a traditional Xbox console.

## What This Post Should Cover

- Which games supported achievements on mobile or handheld-style devices
- Whether they synced back to Xbox Live cleanly
- How many worthwhile games actually existed
- How readers could help expand or maintain a game list

## Good Final Direction

The original note suggests a collaborative game list with contributions via fork and pull request. That is probably the strongest angle for the finished version.
