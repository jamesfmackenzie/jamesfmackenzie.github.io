I used an RGB SCART cable and Micomsoft **SC-512N1-L/DVI capture board. If you apply the right scaling in post-process you can get a pixel perfect image** 😎

  

**Find more details in this YouTube post - I’ve provided process details and product links in the Description:**

  

[https://youtu.be/LBQ5VUifQSg](https://youtu.be/LBQ5VUifQSg)

  

If I were to do this again today, I’d definitely use an OSSC first and then grab 1080p output from there. It provides much more precise capture control e.g. you can tweak things like horizontal sampling rate and sync. You can’t do this with the capture card directly
