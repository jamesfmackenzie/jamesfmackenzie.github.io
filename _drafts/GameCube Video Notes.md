To start

  

Broadly, are two models of GameCube:

  

DOL-001 and DOL-101

  

DOL-001 has a “digital out” port, that will allow you to get the best video quality from your GameCube. If you have this port, you should definitely use it

  

As a side note, the GC outputs better image quality that Wii (in general). So if you are purist c stick with GameCube 

  

A note on resolutions:

  

GameCube is 480i. But a lot of games (especially NTSC games) support progressive / 480p video too.

  

If our game supports 480p, we should definitely use it! But we do need a video solution that supports both.

  

Best quality for GameCubes with digital out:

  

For a true digital to digital solutions, use a GCVideo solution like Carby or Prism.

If you don’t have or don’t want to use an HDMI display, you can use the MK-II, which has a Wii output and you can use a Wii component cable.

  

You can also use either official or unofficial component cables, dlink cables, SCART cables, VGA cables etc (I have these and can talk about them)

  

This official cables does not do sound, so you’ll have a use a simple cable connected to the analog AV port and use it for audio only.

  

Best quality for GameCube *without* digital out

  

If you don’t have a digital out connector on your GameCube, you have to use the analog AV port.

  

If you have a PAL GameCube, you’re in luck! These support RGB output over the analog output. If you buy the official SCART cable, you can get a great picture. In fact this is close to the RGB support you can get from the digital out.

  

The only downside is that this output doesn’t support progressive mode, so only use this option if your GameCube doesn’t have the digital port or your display doesn’t support 480p

  

If you have an NTSC GameCube, bad luck! These do not support RGB over the analog output, you are stuck with crappy svideo! :-(
