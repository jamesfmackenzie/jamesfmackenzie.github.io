The Atari STE looks similar to a regular ST, but it has a handful of meaningful hardware upgrades that some games actually use. This draft is a starting point for a proper roundup of those games, with side-by-side comparisons where possible.

## What Changed on the STE?

Compared with a standard ST, the STE adds or improves a few important capabilities:

- 4096-colour palette
- DMA 8-bit stereo sound, with higher quality sampled audio
- Hardware scrolling
- Blitter support in a more mainstream configuration
- Extra controller support, including Jaguar-style pads in some cases
- SIMM-based RAM upgrades up to 4MB

Not every game uses these features well, and many titles only use one enhancement for a title screen or intro. The interesting part is finding the games where the STE version meaningfully improves the experience.

## Feature Notes

Useful shorthand for the comparison table:

- `ADMA`: audio DMA
- `EPAL`: extended palette
- `HWS`: hardware scrolling
- `BLIT`: blitter usage
- `AJ`: analogue joystick support
