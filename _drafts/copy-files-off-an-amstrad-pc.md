Copying files off an old Amstrad PC can be harder than it should be, especially if the machine has no convenient modern storage interface. This draft is a basic guide to the transfer options that are still realistic.

## Working Approach

The original note says the serial cable route worked well:

- serial cable
- Kermit for transfer
- XMODEM as another option
- Procomm on the PC side

That is already enough to form a useful practical post.
